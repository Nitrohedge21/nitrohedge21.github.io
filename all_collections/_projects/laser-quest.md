**Description**
<br>This project was made on Unreal Engine 5 using a combination of C++ & Blueprints.
This project was created as the final year group project, made for a client. This project
was specifically created for Vive Focus 3 headset and we have incorporated bHaptics vest as
per client's request. This project was aimed to be a framework for the client to use to make
VR arcade shooter styled games. The client was happy about the work we have done and the end
product.
